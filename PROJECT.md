# adk-utils-example

A modern chat application demonstrating the integration of Google Agent Development Kit (ADK) with Next.js 16, AI SDK, and custom utilities.

## Target Audience
Developers looking for a starting point to build AI agents using the Google ADK, specifically integrating with Ollama local models via `@yagolopez/adk-utils`. 

## Core Principles
1. **Performance**: Utilize Next.js 16 App Router and React Server Components.
2. **Architecture**: Separate concerns using `/api`, `/app/agents`, and `/components`.
3. **Design**: State-of-the-art UI with Tailwind CSS 4, smooth animations, and clear typography.

## Key Capabilities
- Multi-model intelligent chat (Gemini & Ollama).
- Tool execution with visual feedback (Mermaid charts, time fetching).
- Advanced chat controls (streaming responses, text formatting, abort generation).
