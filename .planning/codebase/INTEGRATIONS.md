# Integrations

## AI Providers
- **Google Gemini**: Primary cloud-based LLM provider via `@ai-sdk/google`.
- **Ollama**: Local/Self-hosted LLM support via `ollama` and custom `OllamaModel` utility.

## Cloud Services
- **Google Cloud Storage**: Used for persistent file storage (`@google-cloud/storage`).

## External Tools
- **Mermaid**: For diagram generation and rendering.
- **Streamdown**: For rich markdown and code streaming.
