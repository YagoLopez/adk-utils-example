# Tech Stack

## Core Framework
- **Next.js 16.2.1**: React framework for production.
- **React 19.2.4**: UI library.
- **TypeScript 5.9.0**: Static typing.

## AI & Agentic
- **Google ADK 0.6.0**: Agent Development Kit for building autonomous agents.
- **ADK Utils 0.2.7**: Custom utilities for ADK (by @yagolopez).
- **AI SDK (Vercel)**: Core AI SDK for React/Google integrations.
- **Google Generative AI**: SDK for Gemini models.
- **Ollama**: Integration for local LLM execution.

## Styling & UI
- **Tailwind CSS 4**: Utility-first CSS.
- **Radix UI**: Primitive components (Slot, Tooltip).
- **Lucide React**: Icon set.
- **Streamdown**: Markdown rendering with mermaid and code support.

## Development Tools
- **ESLint 9.15.0**: Linting.
- **Prettier 3.8.1**: Formatting.
- **PostCSS**: CSS processing.
